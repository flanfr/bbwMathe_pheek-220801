\section{Algebra}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Füllen Sie die Lücken aus:

$$(2a + 4b)^2 = (4a^2 + ...... + ......)$$
\TRAINER{$4a^2 + 16ab + 16b^2$}

$$(2a + ......)^2 = ...... + ...... + 9b^2$$
\TRAINER{$4a^2 + 6ab + 9b^2$}

$$(...... - xy)^2 = ...... - 6xy + ......$$
\TRAINER{$(3-xy)^2 = 9 -6xy + x^2y^2$}

$$(6  ......) (......) = 36 - k^2$$
\TRAINER{$(6 - k) (6 + k) = 36 - k^2$}

$$(......  + 5b) (......) = 16x^2 - ......$$
\TRAINER{$(4x + 5b) (4x - 5b) = 16x^2 - 25b^2$}
